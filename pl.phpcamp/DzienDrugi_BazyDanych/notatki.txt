1) Wybrać wszystkich mężczyzn urodzonych po 1999 roku (582)
2) Zliczyć ilość klientów każdej płci jednym zapytaniem (male - 1627, female -1641, unknown - 1732)
3) Wybrać osobę z największą liczbą zamówień jednym zapytaniem (id = 978019)
4) Wybrać wszystkich których imię lub nazwisko zaczyna się od litery "T" (434)
5) Zliczyć jednym zapytaniem wszystkie miasta (1284)

1. SELECT COUNT(*) FROM `clients` WHERE `gender` = 'male' AND `date_of_birth` > '2000-01-01';
2. SELECT `gender`, COUNT(*) AS `cnt` FROM `clients` GROUP BY `gender`;
3. SELECT `id`, `name`, SUM(`orders_count`) AS `max` FROM `clients` ORDER BY `max` DESC LIMIT 1
4. SELECT `name`, `surname`, COUNT(*) FROM `clients` WHERE `name` LIKE 'T%' OR `surname` LIKE 'T%'
5. SELECT COUNT(DISTINCT(city)) FROM `clients`
